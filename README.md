
new-tab-panopticon
==================

This extension makes the website [London Panopticon](http://vis.oobrien.com/panopticon/) made by [Oliver Oâ€™Brien](https://github.com/oobrien) appear in the New Tab page of Chrome instead of the regular one.

###Installing

Download the .crx (chrome extension) file from this link:

https://github.com/Gan0k/new-tab-panopticon/raw/master/crx/new-tab-panopticon.crx

and drag and drop it onto the ```chrome://extensions/``` page.
